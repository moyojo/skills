# 修真要素参考

这些要素用于抽取天位、地位、人位。模型的三维分别是：力量来源、承载媒介、作用目的。

真正的核心不是某张固定列表，而是分面分类：先把世界拆成若干可组合分面，再把分面要素放入天位、地位、人位。当前采用扩展 PMEST 八分面：本体、属性、过程、互作、场域、时序、符号、目的；也可随口简称体、性、化、缘、域、时、识、愿。它们是抽样和组合的镜头，不是封闭本体论。

权威池数据在 `data/`：

- `data/pools.md`：八分面、基础要素、释义、细分、主题偏置、条件偏置、低阶降尺度、复合父级概念和分支绑定规则。

修改池子后运行：

```bash
python3 skills/xiuzhen-art-generator/scripts/validate_pools.py
```

选择规则：

- 现有要素已经足够精确时，直接使用。
- 要素对当前法门过宽时，细分一次，例如 `名 -> 真名/封号/咒名`，`数 -> 比例/周期/极限`，`梦 -> 入梦/梦占/梦中修行`。
- 只有当新要素明确属于某个分面，并且仍能通过天地人解释时，才创造同级新要素。
- 范畴可扩展；新增顶层分面必须能与天位、地位、人位组合，而不是只作为装饰标签。
- 复合概念放入 `data/pools.md` 的复合章节，不要混入普通要素池，除非它确实可以作为普通天/地/人要素抽样。

## 范畴

### 体：本体

世界由什么构成。常见要素包括气、形、质、名、数、象、灵、虚、实、器、身。

### 性：性质

它具有什么质地、材料和偏性。常见要素包括性、阴阳、五行、灵材、血脉、毒净、刚柔、寒热。

五行之说在修真世界广泛流传，不是因为五行能解释全部大道，而是因为它最适合作为低中阶修士、坊市、宗门教习和百艺匠人共享的公共语法。五行把难以直接观察的偏性、药性、材性、脉性、阵性和器性压成金木水火土五类，又提供生克、旺衰、相乘、相侮、归经、流转这些可教学、可交易、可诊断、可配伍的关系。它足够粗，粗到练气弟子也能学；又足够有用，能让丹师配伍、器师选材、医家辨证、阵师定方位、剑修借材性和势路说清楚最小构型。也正因多个天位、地位、人位都需要构型而非散乱并列，五行才会被反复使用：它给“多个要素如何相生、相克、相制、相助、相侮、归藏”提供了现成的结构模板。

但五行不是最高本体论。高阶法门可以借五行入门、交易和命名，却不能把一切都还原成五行。五行常是公共解释层、教学层、百艺配伍层；真正的天位动力仍要回到本次抽到的来源、增长、衰竭、失控和度量。

### 化：过程

它如何运动、生成、转化、分合、显隐或逆转。常见要素包括生、长、化、灭、合、分、升、降、藏、显、转、逆。

### 缘：互作

万物如何彼此影响、约束、交换、污染、净化或传承。常见要素包括感应、交换、寄生、共生、支配、契约、召唤、模仿、映照、污染、净化、传承、因果、平衡。

### 域：场域

它在哪里发生。常见要素包括方位、中心、边界、洞天、山川、阵域、界壁、内外、律令、等级。

### 时：序列

它何时发生、按何节律运行。常见要素包括时序、循环、命运、劫数、节律、机缘、岁差。

### 识：认知

它如何被感知、记忆、推演、言说和领悟。常见要素包括观、听、嗅、尝、触、觉、梦、忆、忘、悟、算、言、符。

### 愿：价值

它为何被追求、牺牲、禁忌、承认或号令。常见要素包括欲、情、德、罪、功、愿、信、礼、义、利、权、名。

## 天位说明

天位是被提升为力量来源的要素。必须定义：

- 来源：力量从何而来。
- 增长：什么条件会让力量变强。
- 衰竭：何时力量耗尽或变弱。
- 失控：力量过载时会如何扭曲。
- 度量：修士如何测量或比较它。

## 天位案例类型

好的天位解释会把机制绑定到要素本身。坏的解释要么只是把题材贴进氛围文本，要么把媒介/目的误当成力量来源。

### 案例 1：表面题材不一定是天位

灵植例子：

- 反例：“这是灵植，所以天位是生长。灵植长得越久越强。”问题在于成熟灵植不应因为停止生长就失去力量。
- 正例 A：天位是 `气/灵力`，作为供能来源；`生/长/藏` 是地位。力量来自积累的灵气，年轮和药性结构只是储存与导流的媒介。
- 正例 B：天位是 `藏`，作为核心结构；灵植强是因为未耗尽的药势被封存压缩。最强点在“将熟未熟”；一旦完全采收或暴露，藏势泄尽。
- 正例 C：天位是 `等级` 或 `时序`，作为权威结构；“百年/千年/万年”有意义，是因为年岁标记位阶、周期完成或历法权柄，而不是因为时间只是装饰风味。

规则：判断候选天位是否自身产生力量。如果它只描述外观、材料、阶段或目标，就移到地位或人位。

### 案例 2：同一法术可以有不同天位归属

火系法术可以有不同的真实天位：

- `气/灵力` 为主天位，火为媒介：普通火球。力量随灵力供给增长，火形只是承载。
- `化` 为主天位，火为表现：力量来自状态转化的差值。毒变灰、寒转热这类转化发生时最强。
- `显` 为主天位，火为揭示：火焰暴露隐藏腐败时力量达到峰值；真相显明后力量下降。
- `灭` 为主天位，火为目的：力量由终结、腐朽、湮灭驱动；火只是可见杀伤形态。

规则：不要从特效里选天位，要从会增长、衰竭、失控的力量结构里选。

### 案例 3：复合天位结构

主辅供能只是最简单的结构。复杂法门常会形成一个具象的整体道形，让多个基础天位在其中就位，而不是形成上下层级。

- 主辅结构：`气/灵力` 是供给，`火` 是载体，`灭` 是目的。简单有效，但不足以支撑剑道或成熟道统。
- 整体道形：`剑` 不是压在其他天位之上的上位天位，而是把若干天位共同铸成剑的整体形态，即铸道为剑。这里的总纲只作内部判据，不应原样写进设定正文；生成时要转译成具体剑路、法力运行、局势切分、承压方式和修炼禁忌。
- 纲位结构：在 `剑` 这个整体道形中，六纲不是固定能力，而是六个天位槽位。`剑气` 处理某种力量如何成形，`剑势` 处理某种势如何切出界限，`剑胆` 处理某种意义如何承担反压，`剑心` 处理某种本心如何约束剑性，`剑意` 处理某种意志如何塑造斩击，`剑道` 处理某种自性如何成为锋刃规则。每一纲都要从真实天位中取解释来源，不要把纲名本身当成动力。
- 杀伐倾向：剑修默认比同境界普通修士更强势，且极善杀伐。剑法应天然擅长杀伤、破防、抢线、逼迫选择和终结战斗；若写出同级斗法里“不如普通法术直接”的效果，必须有明确原因，例如残篇、误修、非杀伐用途、被克制、资源不足或修士刻意压剑。不要为了平衡强行给剑法本身添加削弱；剑修的常见难处在修士侧，包括难练、难学、难精，以及极于剑后被剑性六纲反塑，心性、判断、行事和道途走偏锋。
- 纲位数量：剑道六纲是成熟地图，不是必填清单。低阶剑法可能只触及剑气；金丹剑法可能组合剑气、剑势、剑心；顶级剑道才可能统合六纲直到不再彼此分离。`剑` 作为道形统合标记不占普通天位数量，其他天位仍应从完整大池抽样。
- 深浅判定：六纲各自有庸俗层与本质层。低阶、低质或缺少上下文时，剑气可只是属性剑芒，剑势可只是蓄势压迫，剑胆可只是勇气爆发，剑心可只是道心坚定，剑意可只是剑招意境，剑道可只是无形斩。达到境界/质量门槛或被上下文牵引时，才进入更具体的机制层。庸俗层也要受绑定天位影响：剑势绑定 `气` 是气势，绑定 `边界/方位` 是局势或形势，绑定 `时序` 是时势，绑定 `名/权/等级` 是名势、权势或位势，绑定 `数/象` 是数势或象势，绑定 `藏` 才是蓄势。
- 蓄势约束：蓄势不是“前摇长所以废”。若剑势由 `藏/蓄势/养势` 解释，它应成为同境界斗法中的危险信号：对手必须打断、逼其提前出剑、脱离剑线或付出防御资源。只有上下文另有 `大势`、阵禁、山门、峡道、洞府、城池、宗门等固定媒介时，才写成长期布置的地形杀局。
- 正文约束：不要在最终设定集正文中复述整体道形总纲、六纲判据或提示词解释。读者应该看到某门剑法如何练、如何出剑、何时变强、何处反噬，而不是看到世界观大纲被复述。
- 质量：1.00 表示当前境界的正常质量。低于 1 表示粗糙、借来、不稳或低纯度；高于 1 表示精炼、高上限或近道。质量可以超过 1，不是百分比。
- 约束结构：`藏` 与 `显` 只有在法门解释了从隐藏压力到显露的相变时才能共存。否则应把其中一个变成纯化目标或移除。
- 残留结构：被削弱的天位可以作为触发器、度量、禁忌或失败模式保留，而不再作为力量来源。

规则：多个天位看似无关时，要么找到真正能让它们就位的整体道形，要么承认它们只是相加或冲突，并付出代价。不要把清单伪装成大道。数量和质量必须适配境界。

## 按境界决定天位数量与质量

- 练气：通常 1 个粗糙天位，可以随机取用或借势，连贯性较低。
- 筑基：1-2 个天位，开始出现粗略层级。
- 金丹：2-3 个天位，应能看出君臣佐使或主干/纲位/支援关系。
- 元婴：2-4 个天位，结构应能支撑魂魄、远遁、法域、传承或老怪级生存需求。
- 化神：3-5 个天位，整体道形和面向法则的纲位开始重要。
- 炼虚/合体：4-7 个天位，应体现统合、裁剪和高质量纲位。
- 大乘：5-8 个天位，只有到这里才自然出现“铸天为一”或近乎完整的整体道形。

### 案例 4：纯化可能胜过更多力量源

更多天位通常意味着更多可用力量，但也意味着更多增长条件、衰竭点、度量方式和失败模式。

- 纯化前：`感应` 与 `义` 同时参与铸道为剑，使剑道在盟友战中擅长索敌和护誓。
- 高阶问题：感应拖入太多关系、恐惧、誓约、血脉和债务，剑因此迟疑。
- 纯化后：`感应` 降为地位或索敌方式，`义` 被吸收进剑心/剑胆。广度减少，决断更清，斩击更锐。

规则：纯化不是无理由削弱，而是用广度换取连贯性、更少矛盾，以及更严格条件下的更高上限。

## 多要素角色

任一角色都可以包含多个要素。

- 多天位：多个力量源共同供能。强，但要求它们的增长、衰竭、失控和度量能互相换算或互相制衡。
- 多地位：多个媒介共同承载，如剑身、剑势、阵纹、真名、血脉。但它们不能平行摆放，必须分出主承载、传导/触发、锁定/感知、稳态/约束或代价承载等角色，形成媒介链、媒介阵列或君臣佐使。
- 多人位：多个目的同时成立，如攻击还要索敌、封印还要净化、治疗还要诊断。但它们不能写成多个同等独立效果，必须分出主效用、索敌/识别、传递/扩散、约束/封锁或收束/终局等角色，形成一个目的链。

多地位和多人位也要有构型。一个好结构通常能回答：哪个要素是主干？哪个负责把效果送达？哪个负责识别对象？哪个负责限制边界、敌我或反噬？哪个只在收束或失败时生效？如果回答不出来，就不是复合结构，而是松散堆料。

多天位法门并不自动更好。它们可能是：

- 多项式：两个或更多天位彼此相乘，例如 `边界 x 分 x 义` 形成剑道，在义理决断划出清晰界限时增长。
- 相加式：每个天位分别贡献力量，但不彼此精炼。
- 冲突式：一个天位的增长条件会消耗另一个天位，例如 `藏` 需要尚未揭开的压力，`显` 却在揭示时增长。
- 拖累式：弱质或不纯天位提高维护成本，阻挡更高境界晋升。
- 纯化式：移除或削弱一个天位会减少力量源，却让法门更锋利、更稳定或更接近道。

攻击型法门至少要加入一个用于索敌的媒介或目的要素，例如感应、观、听、因果、方位、名。除非该法门故意自指或本就是范围攻击。

## 主题偏置

主题只能改变权重、解释绑定或后处理，不能把随机候选缩成小池。用户要求某个百艺门类时，开放维度仍从完整要素池抽样；主题元素只能提高相关性，不能排除其他要素。剑道尤其不设主题小池：剑法可以自动补 `剑` 作为道形统合标记，但其他天位必须从完整大池随机，六纲再按抽样结果绑定解释来源。

- 丹药食养：生、长、化、质、尝、净化
- 器械工巧：形、质、实、化、名、律令
- 符箓契令：名、言、律令、契约、象、信
- 阵禁界域：方位、边界、中心、循环、律令、平衡
- 医毒蛊疫：性、触、嗅、尝、污染、净化、寄生
- 灵植御兽：生、长、共生、支配、感应、灵
- 魂梦心识：灵、梦、忆、忘、悟、觉、映照
- 鬼神幽冥：灵、罪、愿、信、召唤、传承、契约
- 卜筮星数：数、象、算、因果、命运、方位、时序
- 剑道：不列小池；`剑` 作道形统合标记，其余天位从完整要素池抽样
