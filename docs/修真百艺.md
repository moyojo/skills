什么修真百艺，三千大道，硬写的话很难

不过要是从几个维度入手，每个维度列举几个要素，组合就很多了

比如说三个维度，**动力、媒介、目的**

要素也很好找，从于我们**人类理解世界时反复使用的底层范畴——存在、变化、秩序、关系、认知、价值**这些范畴中找

基于这几个范畴，就可以引出很多要素

> **存在**：气、形、质、性、名、数、象、灵、虚、实 
>
> **变化**：生、长、化、灭、合、分、升、降、藏、显、转、逆 
>
> **秩序**：阴阳、五行、时序、因果、命运、平衡、律令、等级、循环、边界、中心、方位 
>
> **关系**：感应、交换、寄生、共生、支配、契约、召唤、模仿、映照、污染、净化、传承 
>
> **认知**：观、听、嗅、尝、触、觉、梦、忆、忘、悟、算、言 
>
> **价值**：欲、情、德、罪、功、愿、信、礼、义、利、名、权
>

## 存在：世界由什么构成
可以有：

气、形、质、性、名、数、象、灵、虚、实

+ 气：流动的能量、生命力、场、势。
+ 形：结构、形体、器官、阵型、山川脉络。
+ 质：材料、本质、属性、五金百草、骨血皮肉。
+ 性：寒热燥湿、刚柔清浊、善恶毒净等内在倾向。
+ 名：定义、符号、契约、真名、咒名、封号。
+ 数：数量、比例、节律、天机算法、命数。
+ 象：形象、征兆、卦象、星象、梦象。
+ 灵：意识、魂魄、神智、万物有灵。
+ 虚：空、无、间隙、潜能、未显之物。
+ 实：实体、结果、显化、可触可用之物。

## 变化：世界如何运动
可以有：

生、长、化、灭、合、分、升、降、藏、显、转、逆

+ 生：诞生、创造、萌发、孕育。
+ 长：成长、壮大、积累、繁衍。
+ 化：转化、炼化、变形、蜕变。
+ 灭：腐朽、死亡、湮灭、终结。
+ 合：融合、缔结、婚配、炼合、聚众。
+ 分：分解、剖析、离散、裂魂、解构。
+ 升：精炼、飞升、升华、提纯。
+ 降：沉淀、镇压、降服、入世。
+ 藏：隐匿、封存、养势、潜伏。
+ 显：显圣、启示、照见、暴露。
+ 转：轮回、循环、迁移、变轨。
+ 逆：逆天、返祖、回溯、反制、倒因果。

## 秩序：世界按什么规则运行
可以有：

阴阳、五行、时序、因果、命运、平衡、律令、等级、循环、边界、中心、方位

+ 阴阳：对立、互根、转化。
+ 五行：生克、流转、物性系统。
+ 时序：春夏秋冬、昼夜、衰旺、节气。
+ 因果：种因得果、报应、牵连。
+ 命运：定数、变数、劫数、机缘。
+ 平衡：守恒、均衡、调和、代价。
+ 律令：规则、禁制、法度、天条。
+ 等级：尊卑、阶序、品秩、位格。
+ 循环：轮回、周天、潮汐、呼吸。
+ 边界：内外、彼此、封印、界限。
+ 中心：核心、主宰、枢纽、君位。
+ 方位：东西南北、八卦、九宫、星宿定位。

## 关系：万物如何彼此影响
可以有：

感应、交换、寄生、共生、支配、契约、召唤、模仿、映照、  
污染、净化、传承

+ 感应：天人感应、血脉感应、灵犀。
+ 交换：祭祀、买卖、代价、献祭。
+ 寄生：蛊、咒、夺舍、附体。
+ 共生：灵兽、伴生法宝、宗门气运。
+ 支配：御兽、控火、役鬼、掌兵。
+ 契约：誓言、盟约、主仆、天契。
+ 召唤：请神、唤灵、召兽、招魂。
+ 模仿：拟态、变化、傀儡、画皮。
+ 映照：镜、影、梦、分身、替身。
+ 污染：毒、瘟、魔染、诅咒。
+ 净化：祓除、医治、渡化、洗炼。
+ 传承：血脉、师承、碑文、香火。

## 认知：人如何认识世界
可以有：

观、听、嗅、尝、触、觉、梦、忆、忘、悟、算、言

+ 观：看见真相、观星、观气、观人。
+ 听：听风、听心、听命、听天音。
+ 嗅：辨药、寻踪、识妖、闻灾。
+ 尝：尝百草、品灵气、辨毒。
+ 触：摸骨、探脉、感地气。
+ 觉：直觉、预感、灵觉、危机感。
+ 梦：梦占、入梦、梦中修行。
+ 忆：记忆、前世、史书、传承。
+ 忘：忘情、忘我、斩念、洗心。
+ 悟：顿悟、参禅、明心见性。
+ 算：推演、卜筮、阵算、天机。
+ 言：咒语、诗文、律令、真言。

## 价值：万物为何而行
可以有：

欲、情、德、罪、功、愿、信、礼、义、利、名、权

+ 欲：贪求、修炼动力、魔道根基。
+ 情：亲情、爱情、怨恨、执念。
+ 德：功德、仁德、善行、庇佑。
+ 罪：业障、血债、诅咒、刑罚。
+ 功：功业、战功、劳绩、封赏。
+ 愿：宏愿、誓愿、香火愿力。
+ 信：信仰、信任、信物、神道。
+ 礼：祭祀、婚丧、朝拜、宗法。
+ 义：盟义、侠义、道义、因义成道。
+ 利：商道、交易、财货、资源流动。
+ 名：声望、封号、真名、名分。
+ 权：统御、王朝、官职、号令。



这些要素还可以再细分，比如，气分能量、生命力、场、势；形分结构、形体、器官、阵型、山川脉络；生分诞生、创造、萌发、孕育；灭分腐朽、死亡、湮灭、终结；......这些细分后的还能再分，就不细说了

修仙设定里的灵力，只是存在范畴的一个要素的细分的细分，毕竟是仿照电力想象的，基本就是电力的抽象化

假设，我们称呼动力（力量来源）、媒介、目的，为天地人

选择任意不重复要素作为地和人都比较容易想象，比如说通过因果推测名字，通过名字占卜位置，通过视觉消灭敌人，通过方位提升运势

但是要素作为力量来源的天时，是最抽象的，也是最容易搞混的

> 比如说灵植就不是生长作为天位，而是灵力作为天位 因为生长作为天位时，生长的越厉害越有力量，长熟了就没力量了，灵植肯定不能是成熟后就变凡物，所以天位还是灵力，只不过地位里包含了生长和积蓄作为媒介 还经常有小说沿用灵植百年千年万年的设定，这种没有缘由的时间越长越好，可能是藏或者等级作为了天位
>
> 如果价值里拿个罪要素作为力量，可能是业报，煞气什么的
>
> 还有什么香火愿力、文气、功德，闭口禅，算是比较常见的天位
>

如果想用一些稀奇古怪的东西做天位，就得先度量化

**天位，是某个世界要素被提升为力量源泉后的动力学结构。它必须说明力量的来源、增长方式、衰竭条件、失控形态和度量方法**

| **天位** | **真正力量来源** | **最强状态** | **衰竭点** |
| --- | --- | --- | --- |
| **火** | 燃烧中的转化差、显化、毁灭、升腾 | 正在燃、将爆未爆、净化污染时 | 燃料耗尽、灰烬冷却 |
| **名** | 指称、定义、绑定、承认 | 名实相符、万众承认、真名被握住时 | 名不副实、被除名、失传 |
| **数** | 比例、周期、差额、极限 | 达到特定数位、循环闭合、临界点 | 数尽、数乱、比例破坏 |
| **藏** | 未显之物被压缩、封存、蓄势 | 将出未出、压到极限 | 一旦显露，藏势泄尽 |
| **显** | 隐物被照见、真相被揭开 | 揭露瞬间、照见本相时 | 已经明了之后，力量减弱 |
| **契约** | 约束、见证、违约反噬 | 誓成、违约、追责时 | 契解、见证失效、名分崩坏 |
| **梦** | 虚实相叠、心识造境、醒梦边界 | 半醒半梦、梦境稳定时 | 完全醒来或梦境崩散 |
| **愿** | 欲求未竟、信众执念、未来牵引 | 愿未满而众心一致时 | 愿成、愿散、愿被污染 |
| **权** | 承认、统辖、号令、位格 | 众人承认其位、令出必行时 | 失位、叛乱、无人承认 |




总之，选择要素作为天（动力）、地（媒介）、人（目的），就可以构成法门

只选天位，就是天道，不同要素的天位能囊括大部分修炼体系了

只选天地两位，就是所谓的三千大道

三位齐全，就是所谓的万千法门

```plain
一个法门 = 一个要素的天位 × 一个要素的地位 × 一个要素的人位
```

当然了，天地人每位的要素也可以有多个，比如大多数攻击法术都得带有瞄准效果，至少要多一个地和人来达成索敌，不然攻击打自己身上还是太幽默了

**动力、媒介、目的，也可以用来拆解和分析某种法门**

比如小说常见的剑修，剑修的天位肯定不止灵力，因为不只是拼灵力，还拼对剑的理解，拼剑招，拼剑意，剑显然是一个比较复杂、复合的概念：

> 剑，是聚力成锋、分断界限、承担决断、化志为行
>
> 分开此与彼，生与死，进与退，勇与惧，义与暴，身与心、护与伤、名与实、秩序与混乱、守护与杀戮、道义与暴力，持剑者与被剑所指者
>
> 所以，剑做天位，名为剑道，实为铸道为剑
>
> 然后以剑的本质延伸主要元素做剑道的子天位(属性/字段)
>
> 比如说设定，**剑性六纲：剑气、剑势、剑胆、剑心、剑意、剑道**
>
> 剑气考验以何力量束为形体，剑势考验以何局势划分界限，剑胆考验以何意义承担阻力，剑心考验以何本心守持剑性，剑意考验以何意志形塑斩击，剑道考验以何自性自成锋刃
>
> 剑道天位，可以视作六个天位的结合，这么多天位的法门，剑修无疑很强
>
> 这样基本上小说里剑修能装的B都能解释了，属性攻击是剑气，装高手是蓄剑势，爆种是剑胆遇强则强，性格突出道心通明是剑心，剑招能成河成山是剑意形塑斩击，极境能无形斩斩无形是自性成刃
>
> 当然，剑道六天位显然不能乱选，不然选了个生长做剑势，别人等局势你等发育长高，成年人这辈子很难练成了
>

在天位上深挖设定显然比在地和人位深挖设定更精彩，因为首先天位需要想象力，也就能给读者惊喜，其次天位就是力量来源，附加天位的同时，就会带来某种实力的提升，其次天位多了，就容易写出田忌赛马，在某种限制条件下以弱胜强，给主角的机遇带来机会

**然后法门里，那些可职业化、可传承化、可交易化的，能让修士毕生专修，能产出物品或服务，能在宗门坊市中立身的法门各自集中起来，就形成了修真百艺**

这么组合下来，什么法术，百艺，法器，神通，天材地宝，都是比较好想的
